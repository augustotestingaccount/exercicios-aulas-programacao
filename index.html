Comandos mais usados do GIT

  • Clonar repositorio localmente git clone URL
  • Adicionar arquivos para comitar git add .
  • Adicionar UM arquivos para comitar git add ARQUIVO
  • Comitar arquivo(s) git commit -m "MENSAGEM DO COMMIT"
  • Enviar codigo para github git push
  • Para atualizar o codigo local com os do git git pull
